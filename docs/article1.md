# 课程实践：使用开源软件建立个人Blog
## 一、实验任务
1. 创建公开GitHub仓库 57-11.github.io
2. 编写Markdown博客内容，使用MkDocs静态博客工具
3. 配置GitHub Actions，提交代码后自动打包发布到GitHub Pages

## 二、完整操作过程
全程仅使用GitHub网页端操作，无需下载本地软件：
1. 创建专属域名公开仓库 57-11.github.io
2. 新建.github/workflows/deploy.yml 自动化部署脚本
3. 编写mkdocs.yml站点配置，设置博客名称与导航
4. 在docs目录编写首页和第一篇Markdown博客文章
5. 提交所有文件，触发Actions流水线自动构建静态网站
6. 在仓库Settings开启GitHub Actions作为Pages部署源

## 三、实验收获
掌握GitHub仓库网页操作、Markdown写作、CI/CD自动化发布、GitHub Pages静态站点部署流程。
